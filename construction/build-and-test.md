# Build and Test

**Purpose**: Build all units and execute comprehensive testing strategy

## Prerequisites
- Code Generation must be complete for all units
- All code artifacts must be generated
- Project is ready for build and testing

## Steps

1. **Analyze Testing Requirements** - Determine appropriate testing strategy (unit, integration, performance, e2e, contract, security tests)
2. **Generate Build Instructions** - Create build-instructions.md with prerequisites, build steps, troubleshooting
3. **Generate Unit Test Execution Instructions** - Create unit-test-instructions.md
4. **Generate Integration Test Instructions** - Create integration-test-instructions.md
5. **Generate Performance Test Instructions** (if applicable) - Create performance-test-instructions.md
6. **Generate Additional Test Instructions** (as needed) - Contract tests, security tests, e2e tests
7. **Generate Test Summary** - Create build-and-test-summary.md with overall status
8. **Update State Tracking** - Mark Build and Test stage as complete
9. **Present Results to User** - Show completion message with review required
10. **Log Interaction** - Log stage completion in audit.md
