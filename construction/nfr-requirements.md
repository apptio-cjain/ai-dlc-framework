# NFR Requirements

## Prerequisites
- Functional Design must be complete for the unit
- Unit functional design artifacts must be available

## Overview
Determine non-functional requirements for the unit and make tech stack choices.

## Steps to Execute

1. **Analyze Functional Design** - Read functional design artifacts
2. **Create NFR Requirements Plan** - Focus on scalability, performance, availability, security
3. **Generate Context-Appropriate Questions** - Ask about scalability, performance, availability, security, tech stack, reliability, maintainability, usability
4. **Store Plan** - Save as nfr-requirements-plan.md
5. **Collect and Analyze Answers** - Wait for user, carefully review for ambiguities
6. **Generate NFR Requirements Artifacts** - Create nfr-requirements.md, tech-stack-decisions.md
7. **Present Completion Message** - Show review required
8. **Wait for Explicit Approval**
9. **Record Approval and Update Progress**
