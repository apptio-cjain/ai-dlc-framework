# Infrastructure Design

## Prerequisites
- Functional Design must be complete for the unit
- NFR Design recommended (provides logical components to map)

## Overview
Map logical software components to actual infrastructure choices for deployment environments.

## Steps to Execute

1. **Analyze Design Artifacts** - Read functional design and NFR design
2. **Create Infrastructure Design Plan** - Focus on mapping to actual services (AWS, Azure, GCP, on-premise)
3. **Generate Context-Appropriate Questions** - Ask about deployment environment, compute, storage, messaging, networking, monitoring, shared infrastructure
4. **Store Plan** - Save as infrastructure-design-plan.md
5. **Collect and Analyze Answers** - Wait for user, review for ambiguities
6. **Generate Infrastructure Design Artifacts** - Create infrastructure-design.md, deployment-architecture.md
7. **Present Completion Message** - Show review required
8. **Wait for Explicit Approval**
9. **Record Approval and Update Progress**
