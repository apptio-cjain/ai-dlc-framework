# NFR Design

## Prerequisites
- NFR Requirements must be complete for the unit
- NFR requirements artifacts must be available

## Overview
Incorporate NFR requirements into unit design using patterns and logical components.

## Steps to Execute

1. **Analyze NFR Requirements** - Read NFR requirements, understand scalability, performance, availability, security needs
2. **Create NFR Design Plan** - Focus on design patterns and logical components
3. **Generate Context-Appropriate Questions** - Ask about resilience patterns, scalability patterns, performance patterns, security patterns, logical components
4. **Store Plan** - Save as nfr-design-plan.md
5. **Collect and Analyze Answers** - Wait for user, review for ambiguities
6. **Generate NFR Design Artifacts** - Create nfr-design-patterns.md, logical-components.md
7. **Present Completion Message** - Show review required
8. **Wait for Explicit Approval**
9. **Record Approval and Update Progress**
